# Vue 3 + Vite

## 文件目录解析

1. index.html：项目入口文件
2. package.json：管理项目依赖和配置的文件
3. public 目录：放置静态资源，比如 logo 等图片
4. vite.config.js：和 Vite 相关的所有工程化配置
5. src：项目源码目录

## 项目架构

1. 基于 Node.js 生态
2. 工程化工具：使用 Vite
3. 应用开发语言：Vue 3
   1. 数据管理：Vuex
   2. 路由管理：Vue-router
   3. `npm install vue-router@next vuex@next`
4. 开发工具：使用 VSCode （编辑器） + Volar （语法提示工具）
5. 调试工具：Chrome + Dev Tools
